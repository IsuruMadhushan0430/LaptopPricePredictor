# Laptop Price Predictor

This repository contains a machine learning project to predict laptop prices using a dataset and a web interface.

## Project Structure

- `model/`: Contains the dataset (`laptop_price.csv`) and notebook for model building.
- `website/`: Flask application for serving the prediction model.

## Setup

1. Create and activate a Python virtual environment.
2. Install required packages (e.g., using `pip install -r requirements.txt`).
3. Run the Flask app from the `website/` folder: `python app.py`.

## Usage

Train the model using the Jupyter notebook and then use the web interface to input laptop specs and get a price prediction.
